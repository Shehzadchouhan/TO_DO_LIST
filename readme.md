# To-Do List ✅

A simple and modern To-Do List built using **HTML, CSS, and JavaScript**. 🎯

## 🚀 Features:
- Add & Delete Tasks 📝
- Mark Tasks as Completed ✅
- Saves Tasks in **Local Storage** 🔄
- Simple & Modern UI 🎨
- Mobile Friendly 📱

## 🔧 How to Use:
1. Clone this repository  
2. Open `index.html` in your browser 🚀
